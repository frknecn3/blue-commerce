<h2>ABOUT</h2>
<hr/>
An E-commerce website I developed using Next.js and coded with TypeScript in which you can search for existing products, add them to the shopping cart, see the details about the product, view user profiles and see every product they sell as well as upload your own products. Website allows both anonymous users and authenticated users to browse through it, but accessing authorized areas require the user to be logged in which was taken care of by the Next.JS' newer tech App Router. Almost all pages are server-side rendered thanks to Next.js therefore the app has a clear advantage in terms of SEO and end-user experience.
<hr/>

<h2>Used Technologies:</h2>
<ul>
  <li>Next.js (for overall project and routing)</li>
  <li>React.js (for Next.js)</li>
  <li>TypeScript (to handle sent and received data properly)</li>
  <li>JavaScript (for rather easier parts in which using TS is unnecessary)</li>
  <li>Tailwind CSS (For faster CSS development)</li>
  <li>Google Firebase (For all Backend requirements such as posting and fetching user and product data)</li>
  <li>Fuse.js (for fuzzy matches to search items easier)</li>
</ul>
